# Octopus 4

Eine selbstaktualisierende Scriptsammlung für Adobe InDesign, die Workflows im Bereich Satz, Typografie und barrierefreie PDF-Erstellung optimiert.

## Was ist Octopus?

Octopus ist eine umfangreiche Sammlung von InDesign-Scripten, die sich automatisch auf dem neuesten Stand hält. Einmal installiert, aktualisieren sich die Scripte selbstständig und stehen direkt im Scripts-Panel von InDesign zur Verfügung.

Die Scriptsammlung deckt verschiedene Themenbereiche ab:

- **Typografie & Layout**: Grundlinienraster, OpenType-Features, Schriftenverwaltung
- **PDF & Barrierefreiheit (UA)**: Prüfung von Tags, Alt-Texten, Artikeln, Sprachzuweisungen und weiteren Accessibility-Anforderungen
- **Dokumentverwaltung**: Export-Funktionen, Wasserzeichen, Übersatzanzeige
- **Interaktivität**: Lesezeichen, Hyperlinks
- **Workflow-Tools**: Dashboard, Font-Installation, Farbmanagement

## Installation

1. Lade die Datei [Octopus4-Starter.jsx](Octopus4-Starter.jsx) herunter
2. Kopiere die Datei in deinen InDesign Scripts Panel Ordner:
   - **macOS**: `~/Library/Preferences/Adobe InDesign/[Version]/[Sprache]/Scripts/Scripts Panel/`
   - **Windows**: `C:\Users\[Benutzername]\AppData\Roaming\Adobe\InDesign\[Version]\[Sprache]\Scripts\Scripts Panel\`
3. Öffne InDesign und starte das Script über das Scripts-Panel (Fenster > Hilfsprogramme > Skripte)

Beim ersten Start richtet Octopus automatisch alle benötigten Scripte ein und hält diese in Zukunft aktuell.

## Verwendung

Nach der Installation findest du die Octopus-Scripte im Scripts-Panel von InDesign. Die meisten Scripte bieten eine benutzerfreundliche Oberfläche und passen sich an das aktuelle Dokument an.

Das **Dashboard** bietet einen zentralen Überblick über alle verfügbaren Octopus-Funktionen.

## Systemanforderungen

- Adobe InDesign (kompatibel mit aktuellen Versionen)
- Schreibrechte im Scripts Panel Ordner
